---

## Challenges

One of the biggest challenges was making the **Flexbox cards responsive on smaller screens**.  
Initially the cards did not adjust properly on mobile devices.

I solved this by learning how to use:

- `flex-wrap`
- flexible card widths
- responsive layout adjustments

These techniques helped the layout adapt better to different screen sizes.
