## Project Description
This project is a beginner-friendly multi-page portfolio website built using semantic HTML5 and CSS. It introduces basic website structure and navigation.

## Technologies Used
- HTML5
- CSS3
- Git & GitHub

## Features
- Multi-page layout
- Semantic HTML structure
- Contact form with validation
- Deployed using GitHub Pages

## How to Run
1. Clone this repository
2. Open `index.html` in your browser

## Lessons Learned
I learned how to structure a website using semantic HTML and connect multiple pages together.

## Challenges Faced
Understanding how pages link together and keeping the layout consistent across pages.
